欢迎使用Python FTP服务器!

目录说明:
- uploads/   : 上传文件目录
- downloads/ : 下载文件目录  
- public/    : 公共文件目录

支持的用户:
- admin     : 管理员用户 (密码: admin123)
- user      : 普通用户 (密码: user123)
- anonymous : 匿名用户 (无密码)

连接信息:
- 地址: localhost
- 端口: 2121

测试命令:
python ftp_client_test.py --test
python ftp_client_test.py --interactive
